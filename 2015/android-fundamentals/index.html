Google Developers Study Jams são grupos de estudo que funcionam numa série de eventos gratuitos, organizados pela comunidade, para explorar determinados temas. O primeiro jam session é apresentado com a parceria da Udacity e seguirá o currículo do curso Android Fundamentals. Os encontros, organizados por facilitadores certificados pelo Google, serão realizados uma vez por semana.

Pré-requisitos: Este é um curso com requerimentos avançados, direcionado para estudantes com ao menos 3 anos de experiência em progamação Java ou em alguma outra linguagem orientada a objetos (como C++, Objective-C ou Python). Se desejar, o curso Intro to Java da Udacity pode auxiliar a você ficar mais familiarizado com Java (ou te ajudar a fazer uma bela revisão).
